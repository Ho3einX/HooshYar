## Overview

HooshYar AI is a production-ready SaaS backend integrating Machine Learning into a scalable FastAPI-based architecture.

The system is designed to demonstrate:

- Clean Architecture implementation
- ML pipeline integration inside API services
- JWT authentication with role-based authorization
- Containerized infrastructure
- Production-ready deployment workflow
- Scalable and maintainable backend structure

The internal codebase, documentation, and API responses are written in Persian (Farsi) to target the Iranian market.
